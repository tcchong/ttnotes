# Sublime Text

## Code Snippets

```
# Step 1: Create code snippet in Sublime Text 2
Tools > Developer > New Snippet

# Step 2
Add the code snippet whithin <![CDATA[ ]]> in <content> element

# Step 3
Set the trigger name in <tabTrigger> element
```
