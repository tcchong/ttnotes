# Overview

* A JavaScript runtime build on Chrome's V8 JavaScript engine
* single thread
* event-driven
* non-blocking I/O
* cross-platform

# Pros & Cons

|Pros|Cons|
|---|---|
|Asynchronous event drive I/O|Callback hell|
|Universal across client and server|Not suitable for CPU-intensive tasks|
